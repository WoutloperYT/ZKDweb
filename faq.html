Wat voor server is Zarenthia?

Zarenthia Kingdoms is een Minecraft survival server waarin spelers hun eigen koninkrijken kunnen bouwen, gebieden claimen en politieke relaties aangaan met andere spelers.

Welke Minecraft versie gebruikt de server?

De server draait op Java Edition 1.20.x.

Hoe kan ik staff worden

Om staff te worden moet je ee ticket maken in de discord en dan de vragenlijst beantwoorden

Kan ik mijn eigen koninkrijk starten?

Ja. Nieuwe spelers kunnen zich aansluiten bij een bestaand rijk of met genoeg leden een eigen koninkrijk stichten.

Is PvP toegestaan?

PvP is toegestaan tijdens oorlogen, officiële gevechten of wanneer koninkrijken een conflict met elkaar hebben.

Zijn cheats toegestaan?

Nee. X-ray, hacks, exploits en dupe-glitches leiden tot een permanente verbanning.
